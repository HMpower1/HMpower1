## 🌑 Główny Projekt: Shadehack
Zaawansowany klient utility do Minecrafta bazujący na szkielecie **Thunderhack**, przebudowany z naciskiem na maksymalną wydajność, unikalne efekty wizualne i skuteczność na współczesnych anticheatach.

- 🚀 **Optymalizacja:** Usunięty zbędny kod z bazy Thunderhacka dla lepszych FPS-ów.
- 🎨 **Visuals:** Autorskie shadery GLSL zaimplementowane w modułach ESP oraz Chams.
- ⚔️ **Combat:** Szybkie wątki obliczeniowe dla optymalnego działania AutoCrystala.

## 🛠️ Stack Technologiczny & Umiejętności
- **Języki:** Java, GLSL (Shaders), Kotlin
- **Narzędzia:** Mixins, ASM, MCP / Yarn, Gradle, Git
- **Zainteresowania:** Game Hacking, Optymalizacja Kodu, GUI Design, Reverse Engineering
